parallax-scrape
===============

node.js program to Download a forum page from forums.parallax.com and extract posts as plain text.

Install
-------

    $ git clone https://github.com/ZiCog/parallax-scrape.git
    $ cd parallax-scrape
    $ npm install request
    $ npm install htmlparser2


Run
---

    $ node parallax-scrape
